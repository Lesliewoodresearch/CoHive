# Examples

This folder contains reference material and examples of how the CoHive system is used.

## Contents

### Persona Reference Material

**`ad-legend-personas.md`** - Short-form persona prompts for advertising legends

**`ogilvy-vs-bernbach-ad-theory.md`** - Comprehensive reference guide for advertising legend personas including:
- Core theories of advertising for each legend
- What makes a great ad in their view
- Their approach to creativity
- Persona review guidelines
- Voice guidelines
- Scoring rubrics

## Purpose

These files serve as:
1. **Reference Material** - Background theory for persona development
2. **Documentation** - How persona voices are constructed
3. **Examples** - Demonstrating the depth of persona characterization

## Related Files

- **Persona Implementations**: `/data/persona-content/` - Actual persona JSON files used in the system
- **Persona System Docs**: `/data/persona-content/README.md` - How to create and use personas
- **Persona Data Loader**: `/data/personaContentData.ts` - Loads persona content from JSON files

## Usage

These are reference documents only. They are not directly imported by the application code. The actual persona implementations are in `/data/persona-content/` as JSON files.
